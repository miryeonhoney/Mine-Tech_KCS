# K-RISK 산출 방법론

> **K Mineral Risk** 팀 SMART-X · 세종대학교 에너지자원공학과 · 2026-07 기준
> 흩어져 발표되는 광물 지표를 하나의 공급망 위험 점수(0~100)로 융합하는 조기경보 지수.
> 구현: `dashboard_app.py`의 `compute_k_risk()` (1시간 캐시)

---

## 1. 목적과 설계 철학

- 수급안정화지수(광해광업공단), 수입 구조(관세청), 지정학 리스크(뉴스·수출통제), 가격 변동(파생지수)은 **각기 다른 기관이 따로 발표**한다. "그래서 지금 어떤 광물이 위험한가?"에 답하는 단일 지표가 없다.
- K-RISK는 이 4개 신호를 가중합해 **광종당 하나의 점수·등급**으로 만든다.
- 목표는 **가격 예측이 아니라 조기경보**다. 방향(오를지 내릴지)이 아니라 **충격 크기(크게 움직일 취약 상태)**를 사전에 알리는 것.

## 2. 산식

### 정식 4축 (수급안정화지수 제공 6광종: 리튬·니켈·코발트·동·텅스텐·몰리브덴)

```
K-RISK = 0.35 × S(수급불안정) + 0.25 × H(수입집중도) + 0.20 × G(지정학) + 0.20 × V(가격변동성)
```

### 잠정 3축 (나머지 42광종)

```
K-RISK(잠정) = (0.25 × H + 0.20 × G + 0.20 × V) / 0.65     ← 재정규화
```

- 수급안정화지수가 공표되지 않는 광종은 S축을 제외하고 남은 가중을 재정규화한다.
- 화면·API 어디서나 **"잠정" 배지를 명시**한다 (숨기지 않는 것이 원칙).
- 모든 점수는 0~100으로 클리핑.

### 등급

| 점수 | 등급 |
|---|---|
| 0 ~ 39 | 🟢 안정 |
| 40 ~ 69 | 🟡 주의 |
| 70 ~ 100 | 🔴 위험 |

## 3. 축별 상세 정의

### S — 수급불안정 (가중 0.35)

- **정의**: `100 − 수급안정화지수(최신 월)`
- **출처**: 한국광해광업공단 수급안정화지수 월별 CSV (2018-01~, 6광종)
- **왜 최대 가중인가**: 4축 중 유일하게 "한국의 실제 수급 상태"를 직접 측정하는 국가 공인 실측 지표이므로.

### H — 수입집중도 (가중 0.25)

- **정의**: 국가별 수입금액 비중의 **HHI(허핀달-허쉬만 지수)** × 100. `Σ(국가별 비중)²`
- **출처**: 관세청 국가별 광종 수출입 실적 (수입금액 기준)
- **폴백 체계**: 관세청 직접 데이터가 없는 광종(희토류 개별 원소·귀금속 등)은 그룹 무역 스냅샷(`_v2_trade_groups`: 희토류/전략광물/귀금속/우라늄 그룹)으로 대체. 그것도 없으면 중립값 50.0.

### G — 지정학 (가중 0.20)

- **정의**: `min(100, 고위험국 수입 비중(%) + 1위 생산국 가산 20점)`
- **고위험국 집합**: 중국·러시아·콩고민주공화국·미얀마 (수출통제·분쟁 이력 기준)
- **1위 생산국 가산**: 세계 1위 생산국이 고위험국이면 +20점 — 한국이 지금 다른 나라에서 수입하더라도, 글로벌 공급의 뿌리가 고위험국이면 구조적 위험이 있다는 판단. 1위국은 자체 매핑(`K_TOP_PRODUCER`) → USGS MCS(`USGS_DATA`, `_v2_usgs1`) 순으로 조회.

### V — 가격변동성 (가중 0.20)

- **정의**: `min(100, 변동계수(CV) × 250)` — 최근 24개월 광물 가격지수의 표준편차/평균
- **출처**: 광해광업공단 파생 광물가격지수(`mineral_index_data2.json`) — 종합/에너지광물/희소금속/메이저금속 4계열
- **매핑**: 광종 → 지수 계열 (`K_MIDX_GROUP` 전용 매핑 → 분류 그룹 폴백: 비철금속→메이저금속, 희소금속·희토류→희소금속, 에너지→에너지광물, 기타→종합)
- **×250 스케일 근거**: CV 0.4(연 40% 변동)가 상한 100에 닿도록 하는 정규화 계수.

## 4. 축·가중치 설계 근거

### 축 선정 — 국제 표준 방법론의 구성요소

| 축 | 준거 방법론 |
|---|---|
| H (HHI) | 미국 법무부 반독점 심사, **EU 핵심원자재(CRM) 평가**, **USGS 핵심광물 지정**이 공통으로 쓰는 공급 집중도 표준 지표 |
| G (공급국 리스크 가중) | EU CRM이 HHI에 공급국 거버넌스(WGI)를 가중하는 구조와 동형 — 한국 실정에 맞게 "수출통제 이력국 비중"으로 구체화 |
| V (변동계수) | IEA·세계은행 원자재 리스크 분석의 표준 취약성 신호 |
| S | 광해광업공단 공식 지수의 계승(반전) |

→ 새로 발명한 축이 아니라 **검증된 국제 criticality assessment 구성요소를 한국 수입 구조 데이터로 재구성**한 것.

### 가중치 — 설계 판단 + 사후 검증 (튜닝하지 않음)

- 설계 논리: S(0.35) 최대 = 유일한 국내 실측 수급 지표 / H(0.25) = 구조적 위험의 핵심 / G·V(0.20) = 촉발 요인·시장 증상 보조.
- **민감도 분석** (롤링 백테스트 96관측, `backtest_sensitivity.py`):

| 가중 조합 | ρ (사전점수 vs 사후 6개월 \|가격변화\| 순위상관) |
|---|---|
| 채택안 0.35/0.25/0.20/0.20 | **0.230** |
| 균등 0.25×4 | 0.229 |
| 수급지수 단독 | 0.063 |
| ±0.10 섭동 23개 조합 범위 | 0.105 ~ 0.334 |

- 해석: **성능의 원천은 특정 가중치가 아니라 "4축 융합" 구조 자체** (채택안 ≈ 균등 가중 ≫ 단독 지표).
- 섭동 조합 중 ρ=0.334짜리도 존재하지만, 백테스트 성적에 가중을 맞추는 것은 과적합이므로 **의도적으로 하지 않았다.**

## 5. 검증 (3종 + 기각 실험 1종)

### ① 롤링 백테스트 (`backtest_rolling.py`)

- 2021Q1~2024Q4 매 분기, **그 시점까지의 데이터만으로** 소급 산출. 6광종 × 16분기 = 96관측.
- 사후 지표: 이후 2개 분기 평균가 대비 \|변화율\| (조기경보이므로 방향이 아닌 충격 크기).
- 결과: 등급별 평균 충격 **안정 9.9% → 주의 17.6% → 위험 28.0%** (계단식) / '위험(70+)' 6건 전부 ±10% 이상 급변 (급변율 100%) / ρ=0.231, t=2.3, **p<0.05** / 수급지수 단독(0.063) 대비 **3.7배**.

### ② 사건 검증 1 — 우크라이나 침공 (`backtest_ukraine.py`)

- T0=2022-01(침공 직전)로 소급 → 침공 후 2개 분기(2022Q2·Q3) 가격과 순위 대조.
- 결과: **ρ=+0.49**. 사전 2위 리튬 → 상승률 1위(+137.2%), 사전 최하위 동 → 최하위(−11.0%).

### ③ 사건 검증 2 — 미국-이란 충돌 (`backtest_iran.py`)

- T0=2026-01(충돌 2026-02-28 직전)로 소급 → 충돌 후 2개 분기(2026Q2·Q3) 가격 충격과 대조.
- S축은 공표 최신값(2025-03) 사용 — 공표 시차 한계 명시.
- 결과: 충격 크기 **ρ=+0.71**. 사전 1위 텅스텐(76.5 위험) → 충격 1위(−21.4%), 사전 최하위 니켈 → 충격 최하위(−4.2%).
- 참고: 텅스텐 −21.4%는 수출통제발 폭등(1년간 43→153 USD/mtu, +256%) 후 고점 급조정. 조기경보 관점에서 "폭락도 충격"이다 — 고점 장기계약 기업의 손실, 공급 기반 붕괴·의존 심화 위험.

### ④ 방향 시그널 실험 — 검증 후 기각

- 가설: 장기(8분기) 평균 대비 이격도로 폭등/폭락 방향 예측 (과열→조정, 침체→반등).
- 결과: 96관측 중 56콜, **적중률 32%** (동전 던지기 이하). 과열은 더 과열되고(리튬 2021), 침체는 더 침체됨(리튬 2023~24) — 추세 반전 타이밍은 이격도로 예측 불가.
- 결론: **방향 시그널은 탑재하지 않는다.** 검증을 통과한 기능만 싣는다는 원칙. (대안: 예측이 아닌 사실 서술로서의 "과열/침체 국면 라벨"은 도입 가능)

## 6. 한계 (공개 원칙)

1. **잠정 3축의 S축 부재** — 42광종은 국내 수급 실측이 빠져 있어 구조적 위험만 반영. "잠정" 배지로 상시 명시.
2. **수급안정화지수 공표 시차** — 월별 공표가 수개월 지연될 수 있음 (이란 검증에서 2025-03 값 사용).
3. **H·G의 연 단위 갱신** — 수입 구조는 연간 통계 기반이라 연중 급변(수입선 교체)을 즉시 반영하지 못함.
4. **사건 검증 표본 6광종** — ρ=0.49/0.71은 유의성 주장 불가(유의성은 96관측 롤링이 담당). "실전 사례 재현"으로 포지셔닝.
5. **가격 데이터 분기 단위** — 월중 급변은 분기 평균에 희석될 수 있음.

## 7. 확장 로드맵

- 수급안정화지수 공표 광종이 늘어나는 즉시 해당 광종은 **잠정 → 정식 4축 자동 승격** (코드 수정 불필요 — S값 존재 여부로 분기).
- 공개 데이터 확보 시 **시장위험지수·비축량 축을 편입해 4축 → 6축** 고도화 (현재 잔여 가중 재배분 방식 예고를 사이트 산식 하단에 명시).
- 이란 전향 기준선(`baseline_iran_20260717.json`, 2026-07-17 봉인)의 채점 결과 공개.
