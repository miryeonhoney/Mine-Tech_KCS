# 데이터 치환 가이드 (예시 수치 → 실데이터)

> 대본·프롬프트의 "예시 수치"를 실제 공공데이터 값으로 바꾸기 위한 매핑.
> 개발자: 「데이터셋명」을 값 슬롯으로 보고, 아래 데이터셋·필드에서 최신값을 받아 주입.
> 담당자: 새 대본 쓸 때 수치 자리에 「데이터셋명: 어떤 값」을 표시해두면 치환이 쉬움.

## 치환 규칙
- 발언 속 모든 수치는 `값 + [데이터셋명]` 형태로 작성 → 칩이 곧 슬롯.
- 실값이 없으면 수치를 비우고 "확인 필요" 또는 "최신 [통계] 검색 권유"로 둠.
- 가격·지수는 API 실시간, 정책·법령 사실은 팩트 카드(검증된 값) 사용.

| "텅스텐 경계 구간" | 「수급안정화지수_텅스텐」 최신값 | 광해광업공단 |
| "환율이 원화가 끌어올림" | 「환율」「원화 위안화 환율」 당일값 | 재정경제부 / 한국은행 ECOS |

| "돼지 부위 등락" | 「돼지 등급별 경락가격」 최신 경락가 | 축산물품질평가원 |
| "유가·주유소 가격" | 「원유 평균 수입가격」 / 오피넷 평균가 | 관세청 / 석유공사(오피넷 API) |
| "석유 비축 완충" | 「국내석유비축일수」 | 한국석유공사 |
| "가스 현물 변동" | 「산업용 경쟁연료 가격」「유럽 천연가스 상대지수」 | 한국가스공사 |
| "호주 일반광 1위(약 42%)" | KOTRA/무역협회 공급국 비중 | KOTRA / KITA |

## 팩트 카드(고정값 — API 아님, 검증된 사실)
- 자원안보특별법 2025.2.7 시행 / 확보전략 33종·의존도 80→50%·재자원화 2→20% / 중국 수출통제 품목 / IEA 니켈 2035년 85%.
- 이 값들은 변동이 적어 프롬프트에 직접 고정. 변경 시에만 갱신.

## 실시간 연동 API (개발용)
- KAMIS(농산물): kamis.or.kr Open API
- 오피넷(유가): opinet.co.kr Open API
- ECOS(환율·물가): ecos.bok.or.kr/api
- KOMIS(광물): komis.or.kr / 공공데이터포털
- 축산물유통정보: data.go.kr/data/15000577
